# SEO Scanner Welcome Page

Welcome page for SEO Scanner Chrome extension installation.

## Description

This is a single-page welcome site that helps users understand how to pin and use the SEO Scanner Chrome extension after installation.

## Files

- `welcome.html` - Main welcome page with installation instructions

## Usage

Simply open `welcome.html` in a web browser to view the welcome page.

You can also serve it using any static web server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js http-server
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000/welcome.html` in your browser.

## Features

- Step-by-step instructions with visual Chrome browser mockups
- Clear annotations for extension pinning and usage
- Responsive design
- English language interface
